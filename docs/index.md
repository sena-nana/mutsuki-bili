# koishi-plugin-mutsuki-bili 文档

koishi-plugin-mutsuki-bili 是一个基于 [Koishi](https://koishi.chat/) 聊天机器人框架的 **B 站 UP 主推送插件**。

它能够自动监控指定 UP 主的直播状态、动态更新和视频投稿，并将通知推送到 QQ、Discord 等多种聊天平台的指定频道中。

## 核心能力

- **直播监控** — 检测 UP 主开播 / 下播状态变化，实时推送通知
- **动态监控** — 跟踪 UP 主发布的图文动态、转发、视频动态等
- **视频监控** — 监控 UP 主的新视频投稿
- **多平台推送** — 借助 Koishi 生态，支持 QQ（OneBot）、Discord、Telegram 等平台
- **灵活管理** — 群级绑定、按类型订阅、暂停 / 恢复推送
- **扫码登录** — 通过 Web 控制台或聊天指令扫码登录 B 站账号

## 注意事项与限制

- 本插件使用的 B 站 API 为**非公开接口**，可能随 B 站更新而变更或失效
- WBI 签名算法为逆向工程产物，B 站更新签名方案后可能需要同步更新
- 建议轮询间隔不低于默认值（直播 1 分钟、动态 2 分钟、视频 5 分钟），过短的间隔可能触发 B 站限速
- 认证 Cookie 存储在数据库中，请注意数据库的访问安全
